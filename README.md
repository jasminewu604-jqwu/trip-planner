# HK Trip Planner 🇭🇰

A shared interactive trip planner built for Ingrid & Arnaud's 
Hong Kong adventure, 1–3 May 2026.

## What it does
- Browse and add activities and restaurants to a shared itinerary
- Upvote and save favourites
- Real-time sync between travellers — comments, upvotes and 
  trip dates update live
- AI trip assistant powered by Claude
- Departure day automatically buffers 3 hours before your flight

## Built with
- HTML / CSS / JavaScript (single file, no framework)
- Supabase (shared database + real-time sync)
- Anthropic Claude API (AI chatbot)
- Hosted on GitHub Pages

## How to update
Replace index.html with the latest version from Claude.
